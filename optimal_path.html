Optimal Pathing System

How to Use:

  1. Upload an image using the "Choose Image" button above
  2. Click on the image to set the START point (green marker)
  3. Click again to set the END point (red marker)
  4. Click "Find Optimal Path" to compute the shortest path
  5. The path will be displayed in cyan on the result image

Note: You can reset points by clicking "Reset Points" and selecting new locations.
